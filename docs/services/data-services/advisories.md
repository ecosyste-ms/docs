Access unified vulnerability data from multiple sources including package registries, software repositories, and vulnerability databases, giving you a comprehensive view of security threats across your entire open source stack.

## Use this service to...

* Access unified vulnarability data for multiple sources
* Identify which packages and projecs are affected by security advisories
* Discover which packages are built from a single repository
* Verify package contents match what's in the source repository
* Identify when security patches were actually committed
